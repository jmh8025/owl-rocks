# Cliff Owl

부엉이가 절벽 아래로 떨어지며 방해 캐릭터들을 피하는 Canvas 웹게임 프로토타입입니다.

## 실행

이 폴더에서 `index.html`을 브라우저로 열면 바로 실행됩니다.

로컬 서버로 실행하려면:

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

접속 주소:

```text
http://127.0.0.1:5173/index.html
```

## 현재 구현

- 마우스/터치 위치를 따라 부엉이 이동
- 절벽 낙하 연출과 바람선 배경
- 여러 방해 캐릭터 스폰
- 충돌 판정, 거리, 점수, 최고 점수 저장
- Q/W/E/R 스킬 슬롯 UI와 입력 반응
- 데스크톱/모바일 반응형 HUD

## 온라인 확장 메모

나중에 io 스타일 온라인 게임으로 확장하려면 아래 순서가 좋습니다.

1. `game.js`의 입력 상태, 월드 상태, 렌더링 함수를 더 작은 모듈로 분리
2. 서버에서 방해 캐릭터 스폰 시드와 시간축을 관리
3. 클라이언트는 마우스 위치와 Q/W/E/R 입력만 서버로 전송
4. 서버는 충돌/점수 판정을 권위 있게 처리
5. 다른 플레이어는 같은 절벽 안의 부엉이 실루엣이나 랭킹으로 표시

현재 코드에서는 게임 판정용 랜덤과 시각 효과용 랜덤을 분리해두어, 추후 서버 동기화로 옮기기 쉽습니다.
